# Zaheer — AI classroom monitoring (FYP)

Final-year project: **AI-Based Classroom Monitoring System** (I3D + LSTM, summaries, highlights, punctuality, alerts).

## Frontend (React)

```bash
cd frontend
npm install
npm run dev
```

- **Build:** `npm run build` (outputs `frontend/dist/`)
- **Demo login:** `admin` / `admin123` (local session until API is wired)

See [`frontend/README.md`](./frontend/README.md) for tooling and `src/` layout.
